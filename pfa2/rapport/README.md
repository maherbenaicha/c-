# Rapport (LaTeX)

## Compiler

- Pré-requis: une distribution LaTeX (TeX Live / MikTeX).
- Fichier principal: `main.tex`

Compilation (exemple):

```bash
pdflatex main.tex
pdflatex main.tex
```

## Images

Les images sont attendues dans `rapport/images/`.  
Le document compile même si certaines images sont absentes (un cadre “Image manquante” sera affiché).
